# House Price Prediction using Linear Regression

## Overview
This project implements a Linear Regression model to predict house prices based on:

- Living Area (Square Footage)
- Number of Bedrooms
- Number of Full Bathrooms

The model is built using Python and Scikit-learn as part of the SkillCraft Technology Machine Learning Internship.

---

## Dataset

Dataset used:
House Prices - Advanced Regression Techniques (Kaggle)

Features used:
- GrLivArea
- BedroomAbvGr
- FullBath

Target:
- SalePrice

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
