# Mall Customer Segmentation using K-Means Clustering

## Overview

This project applies the K-Means clustering algorithm to segment mall customers based on their annual income and spending score.

The objective is to identify groups of customers with similar purchasing behavior, which can help businesses design targeted marketing strategies.

---

## Dataset

Dataset: Mall Customers Dataset

Features used:

- Annual Income (k$)
- Spending Score (1-100)

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## Steps Performed

- Loaded the dataset
- Performed Exploratory Data Analysis
- Checked dataset information
- Checked missing values
- Calculated summary statistics
- Selected relevant features
- Used the Elbow Method to determine the optimal number of clusters
- Applied K-Means Clustering
- Visualized customer segments

---

## Output

The model grouped customers into clusters based on income and spending score.

The resulting visualization highlights different customer segments for business analysis.
